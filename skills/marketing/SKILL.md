# Marketing Skill for Theme & Plugin Agency

## Instructions

When creating marketing content:

### 1. Product Descriptions
- Lead with the main benefit, not features
- Use power words (effortless, powerful, stunning, seamless)
- Include social proof elements
- End with clear call-to-action

### 2. Feature Highlights
- Transform features into benefits
- Use "so you can..." or "which means..."
- Include use cases and examples
- Compare with competitors subtly

### 3. Social Media Posts

**Twitter/X:**
- Hook in first line
- 1-2 key benefits
- Emoji for visual breaks
- Link + hashtags

**LinkedIn:**
- Professional tone
- Problem → Solution format
- Include stats if available

**Facebook/Instagram:**
- Visual-first approach
- Storytelling angle
- Community engagement

### 4. Email Campaigns

**Launch Emails:**
```
Subject: [Product] is here — [Key Benefit]

Hi [Name],

[One-line hook about the problem]

Today we're launching [Product] — [one-sentence description].

Here's what you get:
• [Benefit 1]
• [Benefit 2]
• [Benefit 3]

[CTA Button]

[Urgency/scarcity if applicable]
```

**Update/Changelog Emails:**
- Lead with biggest improvement
- Group changes by category
- Thank users for feedback

### 5. Landing Page Copy

**Structure:**
1. Hero: Headline + subheadline + CTA
2. Problem: Pain points
3. Solution: Your product
4. Features: With benefits
5. Social proof: Testimonials, logos, stats
6. Pricing: Clear options
7. FAQ: Objection handling
8. Final CTA: Urgency

### Tone Guidelines
- Confident but not arrogant
- Technical when needed, simple when possible
- Focus on outcomes, not just outputs
- Use "you" more than "we"

### SEO Considerations
- Include target keywords naturally
- Write compelling meta descriptions
- Use headers strategically
- Include internal links
